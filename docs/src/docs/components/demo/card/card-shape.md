The Card Component can have 2 kind of shape `rounded` or `sharp`

::: raw

<ClientOnly>
  <CardShape />
</ClientOnly>

:::

::: details Code

```vue
<template>
  <base-card title="Rounded" shape="rounded">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
  </base-card>

  <base-card title="Sharp" shape="sharp">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
  </base-card>
</template>

```

:::
